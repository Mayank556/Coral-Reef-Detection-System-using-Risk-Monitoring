# Coral-Reef-Detection-System-using-Risk-Monitoring

> Automated Coral Reef Health Classification using a Tri-Stream Deep Learning Architecture

## Problem

Manual coral reef monitoring is slow, limited, and not scalable. This system uses AI to automatically analyze underwater images and detect coral health status.

## Architecture

**Tri-Stream Hybrid Model:**
- **Spatial Stream** — ResNet50 (texture & structure detection)
- **Contextual Stream** — ViT-B/16 (global colony morphology)
- **Spectral Stream** — SpectralNet CNN on LAB color space (color anomaly detection)

Streams are fused via **Gated Fusion** (learnable α, β, γ weights) → MLP Classifier → 4 classes

## Classes

| Class | Description |
|---|---|
| Healthy | Normal pigmentation, visible polyp activity |
| Bleached | Total zooxanthellae loss, white skeleton |
| PartiallyBleached | Patchy pigment loss, mixed coloration |
| Dead | Structural degradation, algal overgrowth |

## Project Structure

```
├── models/
│   ├── model.py              # CoralVisionNet tri-stream architecture
│   └── fusion.py             # Gated Fusion module
├── training/
│   ├── train.py              # Training pipeline (CLI)
│   └── loss.py               # Focal Loss
├── utils/
│   ├── preprocessing.py      # CLAHE + Gray-World underwater correction
│   ├── inference.py          # MC-Dropout uncertainty estimation
│   └── explainability.py     # Grad-CAM++, Attention Rollout, Spectral Maps
├── evaluation/
│   └── eval.py               # Metrics, confusion matrix, ECE
├── Merged_Coral_Dataset/     # Dataset (train/val/test splits)
├── trained_models/           # Saved weights (generated by training)
├── CoralVisionNet_Training.ipynb  # Training notebook
└── requirements.txt
```

## Quick Start

```bash
# Install
pip install -r requirements.txt

# Train (CLI)
python training/train.py --epochs 25 --batch_size 16 --mode full

# Or use the Jupyter notebook
# Open CoralVisionNet_Training.ipynb in VS Code / Jupyter
```

## Dataset

**Merged_Coral_Dataset** — 45,632 underwater coral images across train/val/test splits.

> ⚠️ PartiallyBleached class has <1% representation (390 images). Handled via Focal Loss + WeightedRandomSampler + 3× weight boost.
